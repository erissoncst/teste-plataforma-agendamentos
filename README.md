# Testes E2E - Plataforma de Agendamentos

Testes E2E com Playwright para o fluxo de agendamento.

## Instalação

```bash
npm install
npx playwright install chromium
```

## Configuração

```bash
cp .env.example .env
```

## Executar Testes

```bash
npm test
npm run test:headed
npm run test:debug
npm run test:ui
npm run test:agendamento
npm run test:report
```
